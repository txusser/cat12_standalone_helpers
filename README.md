# TOOLS - CAT12 Standalone Helpers

Helper scripts for running CAT12 brain segmentation in batch mode on neuroimaging cohorts.

## Scripts

- **prepare_data_for_processing.py**: Organizes and copies T1 NIfTI files into the directory structure expected by CAT12 standalone
- **run_cat12_segmentation_parallel.py**: Runs CAT12 standalone segmentation in parallel across multiple subjects

## Requirements

- Python 3.8+
- CAT12 standalone installation
- MATLAB Runtime (MCR)
